# uni-app图片分包

# 原因

uni-app在打包的过程中图片不会被压缩，并且图片会被打包到主包，非常占用主包内存资源，所以进行图片分包的步骤

# 1.首先开启分包优化

在manifest.json文件(很好奇为啥不是main而是mani)源码视图中,"mp-weixin"中配置以下代码

	"optimization":{"subPackages":true}

# 2.在分包跟目录中创建static目录用于放置静态资源

# 3.打包后效果

成功打包到分包中，并且不占用主包内存
